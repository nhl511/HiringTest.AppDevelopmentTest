# App Development Test

## Installation

1. Clone the repository:

   ```sh
   git clone https://github.com/nhl511/HiringTest.AppDevelopmentTest
   cd HiringTest.AppDevelopmentTest
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

## Usage

### Development

Run the project in development mode:

```sh
npm run dev
```
